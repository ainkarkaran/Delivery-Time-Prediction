# Delivery-Time-Prediction
The Linear Regression model to predict the estimated delivery time (in minutes) based on the parameters like , weather , delivery experience , distance, weather , traffic , vehicle type and preparation time.
<br>
Dataset:-https://www.kaggle.com/datasets/denkuznetz/food-delivery-time-prediction
<br>
<br>
Steps to run the program :- 
<br> 
Start the server by command "uvicorn server:api --reload"<br>
Open the app.html file with browser<br>
Adjust the input values , the result will be displayed
